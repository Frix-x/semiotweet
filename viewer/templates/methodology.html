Méthodologie

Comment les tweets sont traités et les sujets analysés.


D'où proviennent les données et que contiennent-elles ?

Toutes les données, stockées comme récupérées à la volée proviennent de l'API de Twitter.
Les données récupérées contiennent de nombreuses informations concernant les candidats.
De toutes ces informations, nous ne gardons que :

  • l'identifiant du candidat ;
  • son nom ;
  • son pseudo ;
  • la date de création de son compte ;
  • la participation de contributeurs au post de tweets ;
  • si le compte est vérifié ou non ;
  • la description du profil du candidat ;
  • le nombre de followers ;
  • l'url de l'image de profil ;
Au niveau des tweets, il y a encore plus d'informations. Parmi celles-ci, nous gardons :
  • son identifiant du tweet ;
  • le candidat qui l'a posté ;
  • son contenu textuel du tweet ;
  • sa date de publication ;
  • s'il répond ou non à un autre tweet (dans ce dernier cas on stocke aussi l'identificateur du tweet initial et l'identificateur de son auteur) ;
  • le nombre de likes ; ;
  • son nombre de retweets; ;
  • la plateforme à partir de laquelle il fut posté ;
  • sa langue (basée sur la reconnaissance de Twitter) ;
